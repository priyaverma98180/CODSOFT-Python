# CODSOFT-Python
1. Calculator
 A simple Python calculator that performs basic arithmetic operations such as addition, subtraction, multiplication, and division.
2. Password Generator
 A Python application that generates strong and secure random passwords based on the user-defined length.
3. Contact Book
 A Python contact book application that allows users to add, view, search, update, and delete contacts through a command-line interface.
4. Rock Paper Scissors Game
 A Python command-line game where the user plays Rock, Paper, or Scissors against the computer. The program randomly selects the computer's move, compares both        choices, determines the winner according to the game rules, and displays the result. It also allows users to play multiple rounds.
